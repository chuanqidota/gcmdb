# 管理规范

## 模型分组
- **创建**：英文名和中文名唯一。
- **更新**：不可修改英文名。
- **删除**：分组下不能有模型。

## 模型
- **创建**：中文名和英文名唯一。
- **更新**：不可修改英文名。
- **删除**：模型关联、字段关联、实例关联、实例均不存在 >> 删除模型字段、字段分组及模型。

## 模型字段分组
- **创建**：模型ID + 分组名称联合唯一。
- **删除**：分组下不能有字段。

## 模型字段
- **创建**：模型ID + 字段英文名联合唯一，模型ID + 字段中文名联合唯一 >> 异步为所有实例添加字段。
- **更新**：模型ID + 字段英文名联合唯一，模型ID + 字段中文名联合唯一。
- **删除**：字段关联不存在 + 唯一字段不存在 >> 删除字段 + 异步为所有实例移除字段。

## 模型字段唯一
- **创建**：模型对应实例不存在 + 记录唯一。

## 模型字段关联
- **创建**：记录唯一 >> 异步创建实例关联（根据字段关联 + 实例确定关联数据）。
- **删除**：异步删除实例关联。

## 实例
- **创建**：必须字段 + 删除冗余字段 + 补充默认字段 + 验证唯一性。
- **修改**：删除冗余字段 + 补充字段 + 验证唯一性 >> 生成JSON数据 >> 修改实例。
