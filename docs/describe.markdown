# 模型管理规范

## 模型分组

### 创建
- **英文名**和**中文名**必须唯一。

### 更新
- **英文名**不可修改。

### 删除
- 模型分组下不能有模型。

## 模型

### 创建
- **中文名**和**英文名**必须唯一。

### 更新
- **英文名**不可修改。

### 删除
- 模型关联不存在 + 字段关联不存在 + 实例关联不存在 + 实例不存在 >> 删除模型字段 + 删除模型字段分组 + 删除模型。

## 模型字段分组

### 创建
- **模型ID** + **分组名称**联合唯一。

### 删除
- 分组下不能有字段。

## 模型字段

### 创建
- **模型ID** + **字段英文名**联合唯一 + **模型ID** + **字段中文名**联合唯一 >> 异步将所有实例上增加字段。

### 更新
- **模型ID** + **字段英文名**联合唯一 + **模型ID** + **字段中文名**联合唯一。

### 删除
- 模型字段关联不存在 + 模型唯一字段不存在 >> 删除模型字段 + 异步将所有实例上删除该字段。

## 模型字段唯一

### 创建
- 模型对应的实例不存在 + 记录唯一。

## 模型字段关联

### 创建
- 记录唯一 >> 异步创建实例关联（根据模型字段关联 + 实例确定实例关联数据）。

### 删除
- 异步删除实例关联。

## 实例

### 创建
- 必须字段 + 删除冗余字段 + 补充默认字段 + 验证唯一性。

### 修改
- 删除冗余字段 + 补充字段 + 验证唯一性 >> 得到JSON数据 >> 修改实例。
